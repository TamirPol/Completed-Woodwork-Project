Joineries

My Second Woodworking Project

After finishing my first project, the barbeque scraper, it is time to move onto something harder. Joineries. In this project I have cut multiple different joints which include: the butt joint, half lap joint, miter joint, 2 dovetail joints, and the mortise and tenon joint into one woodpiece as shown in the sketchup image. The project itself has no real use, unlike the barbeque scraper which is used to clean the grill, this project's only purpose is to teach me how to create joineries and to advance my woodworking skills.
One thing that I have thought I could do with this project is create a picture frame by putting a board at the back, painting all the pieces, gluing them together, adding the pictures, and putting a glass board on the front.
